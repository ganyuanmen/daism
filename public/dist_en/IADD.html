    # IADD网络

    uToken 是道易程的通用数字货币，可用于与各类由 Smart Common 发行的代币进行兑换。它充当了道易程内部各种代币的价值提现媒介。

    # IADD网络兑换种类

    • ETH 兑换 utoken
    • ETH 兑换 token
    • utoken 兑换 token;
    • token 兑换 utoken;
    • token 兑换 token。

    # Smart Common 代币价值

    代币单价的单位是vita, 表示为1代币相当于多少utoken

    # 授权管理

    道易程提供两种授权方式：第一种是将 uToken 授权给 IADD 网络，允许 IADD 网络对用户的 uToken 数字货币进行操作；第二种是将 token 授权给 IADD 网络，允许 IADD 网络对用户的 token 数字货币进行操作。见下图
