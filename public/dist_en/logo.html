    # logo管理

    Smart Common 的logo可以是（svg、jpg、jpeg、png、gif、webp）格式的图片，也可以是这些图片进行压缩后的 zip 文件。上传的图片文件必须带有明确的后缀名。为了优化链上空间利用并降低 gas 消耗，建议在上传前将图片进行 zip 压缩。在 zip 压缩包中，只允许包含一张图片；如果包含多张图片，系统将随机选择一张作为logo。请注意，由于链上的限制，上传的图片文件或 zip 文件大小不得超过10KB。

       let file = event.currentTarget.files[0]; //获取选择的图片文件

   reader.addEventListener('loadend', function (e) { makeImg(e, type[0]); });
   reader.readAsBinaryString(file);

        }
