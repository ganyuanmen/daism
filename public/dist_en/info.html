# 概述

Smart Common 可以在道易程网站上注册一个社区帐号。该帐号具备发布新闻、进行内部讨论以及组织线上线下活动的功能。此网站支持 activity 协议，其他支持该协议的软件或网站能够订阅社区帐号的最新动态。网站会主动向订阅用户推送相关信息，成为一种宣传的有效途径。

社区功能专为 Smart Common 成员提供，非成员可提交申请加入某个社区。一旦经管理员批准，便可获得发布相关讨论或新闻的权限。

非 Smart Common 成员可通过订阅，在第三方软件或网站（支持 activity 协议）上参与特定讨论或进行回复。
