    # 提案管理

    提案是由 Smart Common 的管理员发起的一种行为，旨在协调成员对于 Smart Common 管理事务的共同决策。提案的流程包括以下步骤：

    # 提案发起：

    Smart Common 的管理员提交有关管理事务或决策的提案。

    # 投票决定：

    Smart Common 的成员有权对提案进行投票，以决定是否执行该提案。

    提示

    在 Smart Common 上，只允许存在一个活跃的提案，即在当前提案未完成之前，不能发起新的提案。

    提案创建之后，需要到 “提案管理”页面进行投票，见下图：

    成员可以投赞成票或反对票，也可以查看提案的内容及当前投票情况，见下图

    投票规则

    投赞成票：当投票率达到设定的通过率之后，提案会自动执行。在冷却时间过后，可以建新的提案。

    投反对票：当反对票使提案无法达到通过率之后，提案会自动作废。在冷却时间过后，可以建新的提案。
